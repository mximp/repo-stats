# Repository statistics

Collects various repository statistics such as files count by type or lines of code.

## Installation

## Usage

As Python module:
```shell
python -m stats <path_to_a_folder> 
```

Output example:
```shell
Repo: /Users/scoritt/prj/repo-stats/repo-stats
Inclusions: ['.py', '.md']
Exclusions: []
Files total: 2
LoC total: 56
Extensions: {'.py': (3, 56)}
```
