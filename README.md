# Repository statistics

Collects various repository statistics such as files count by type or lines of code.

## Installcation

## Usage

As Python module:
```shell
python -m repo_stats <path_to_a_folder> 
```

Output example:
```shell
Repo: \users\repo
Total files: 100
Total LoC: 1234235
Extensions: {
  '.xml': (233, 12341),
  '.css': (232, 1231)
}
```
